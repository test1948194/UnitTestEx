отчет по тестированию проекта FileStorage
цель работы была проверить корректность методов добавления удаления и поиска файлов с помощью фреймворка NUnit  и исправить все найденные ошибки для успешного прохождения тестов
большую часть времени заняло разбирательство с структурой проекта и наименованием папок было несколько дублирующихся папок UnitTestEx и FileStorage из-за чего проект не собирался и выдавал ошибки путей пришлось более внимательно изучить всю иерархию и исправить ссылки в файле FileStorageTests.csproj
в ходе анализа кода найдено и исправлено множество ошибок в классах File и FileStorage 

в классе File исправлено

деление длины контента на 2.0 для типа double 

добавлена проверка на наличие точки перед получением расширения 




в классе FileStorage убраны пробелы в именах методов

исправлена инициализация доступного размера

заменен метод Contains на Equals для точного поиска файлов

исправлено полное удаление списка файлов в тестах удалена смесь MSTest и NUnit оставлен только NUnit


добавлено три новых теста проверка файла с расширением, проверка файла с пустым контентом и проверка расчета размера 
все тесты успешно прошли изменения, ура
